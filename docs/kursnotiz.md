# Kursnotiz

Diese Datei wird durch eine GitHub-Action automatisch verarbeitet.

## Ablauf

1. Markdown-Datei ändern.
2. Commit erstellen.
3. Push nach GitHub.
4. Pipeline beobachten.
5. PDF-Artefakt herunterladen.

## Merksatz

Eine Pipeline beschreibt wiederholbare Arbeitsschritte.

In diesem Beispiel:

```text
Markdown prüfen
-> HTML bauen
-> PDF bauen
-> Ergebnis als Artefakt bereitstellen
```
